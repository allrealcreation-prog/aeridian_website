# AERIDIAN Engineering Roadmap

## Phase 20: Luxury Motion & Micro-Interactions (Next)
- Activate Magnetic buttons with physics-based spring return across all primary CTAs.
- Implement staggered, eased text-reveal animations for all typography (`y: 50, opacity: 0` -> `y: 0, opacity: 1`).
- Enable Parallax scrolling on lifestyle imagery.
- Construct smooth page transitions (SPA-like) utilizing the existing `GlobalTransitionOverlay`.

## Phase 21: Deep WebGL Immersion
- Re-activate the 3D Object rendering within the Hero Cinematic section.
- Implement particle flows or atmospheric distortion effects in the background.
- Integrate IntersectionObservers to dynamically pause WebGL rendering when the canvas leaves the viewport.

## Phase 22: Conversion Rate Optimization (CRO)
- Implement predictive search with image thumbnails directly in the header overlay.
- Create an intelligent "Upsell / Cross-sell" module dynamically injected inside the Cart Drawer.
- Introduce dynamic localized pricing and "Free Shipping Progress" bars based on cart totals.

## Phase 23: Headless Preparation
- Decouple all remaining Liquid-rendered JSON dependencies into explicit REST / GraphQL Storefront API fetches.
- Finalize Service Worker offline caching for 3D assets.
