# Security Policy

## Supported Versions

Currently, only the `main` branch (production v1.0.0+) receives security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Security is a primary priority for the AERIDIAN theme. If you discover a vulnerability, please do NOT open a public GitHub issue.

Instead, please email the security team at **security@aeridian.example.com**. 
Include the following information in your report:
- Type of vulnerability (e.g., XSS, CSRF)
- Steps to reproduce
- Potential impact
- Any suggested remediations (if known)

We will acknowledge receipt of your vulnerability report within 48 hours and strive to provide a timeline for a fix.
